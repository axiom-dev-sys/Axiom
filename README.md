# Axiom Engine

Axiom Engine — собственный 2D/3D игровой движок, написанный на C++ с использованием OpenGL.

Проект создаётся с целью изучения внутренней архитектуры игровых движков и разработки собственной игровой системы с упором на ECS, рендеринг и игровую логику.

## Текущие возможности

На данный момент движок находится в ранней стадии разработки и умеет:

Рендерить сцены через OpenGL;
Работать с текстурами;
Базовую систему управления ресурсами;
Архитектуру ECS (Entity-Component System);
Простейший игровой слой (Game Layer);
Демонстрационную сцену с базовой игровой логикой;

Движок уже способен запускать простую сцену, которая демонстрирует текущую архитектуру и рендеринг.

## Технологии:
C++;
OpenGL;
GLFW;
GLAD;
stb_image;

## Цель проекта
Основная цель Axiom Engine — не просто создать движок, а понять, как он устроен внутри.

Проект развивается вокруг трёх направлений:
## Геймдизайн — создание ощущаемого игрового опыта;
## Системная архитектура — ECS, логика, баланс и расширяемость;
## Сюжет и атмосфера — глубокие и осмысленные игровые миры.

Движок разрабатывается под будущие собственные игры с уникальной подачей и механиками.

## Сборка и запуск
1. Перейдите в GitHub-репозиторий:
axiom-dev-sys
2. Скачайте проект или клонируйте репозиторий
3. Откройте проект в Visual Studio
4. Выберите конфигурацию:
Debug (рекомендуется для разработки);
Release (для тестирования производительности)
5. Соберите и запустите проект

## Текущее состояние
Сейчас движок находится в активной разработке.

Основной фокус:
- стабилизация архитектуры
- исправление технических ошибок
- расширение базовых систем

После этого этапа начнётся переход к более сложному игровому контенту и механикам.
