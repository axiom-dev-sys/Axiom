# 🧠 Axiom Engine

Axiom Engine — собственный 2D/3D игровой движок, написанный на C++ с использованием OpenGL.

Проект создаётся с целью изучения внутренней архитектуры игровых движков и разработки собственной игровой системы с упором на ECS, рендеринг и игровую логику.

## ✨ Текущие возможности

На данный момент движок находится в ранней стадии разработки и умеет:

- Рендеринг сцен через OpenGL
- Работа с текстурами
- Базовая система управления ресурсами
- Архитектура ECS (Entity Component System)
- Простейший игровой слой (Game Layer)
- Демонстрационная сцена с базовой игровой логикой

Движок уже способен запускать простую сцену, демонстрирующую текущую архитектуру и рендеринг.

## 🧰 Технологии
- C++
- OpenGL
- GLFW
- GLAD
- stb_image
## 🎯 Цель проекта

Основная цель Axiom Engine — не просто создать движок, а понять его внутреннее устройство.

Проект развивается вокруг трёх направлений:

- 🎮 Геймдизайн — создание ощущаемого игрового опыта
- ⚙️ Системная архитектура — ECS, логика, баланс и расширяемость
- 📖 Сюжет и атмосфера — глубокие и осмысленные игровые миры

Движок разрабатывается как основа для будущих собственных игр с уникальной подачей и механиками.

## 🛠️ Сборка и запуск
- Перейдите в GitHub-репозиторий: axiom-dev-sys
- Скачайте или клонируйте проект
- Откройте проект в Visual Studio
- Выберите конфигурацию:
Debug (рекомендуется для разработки)
Release (для тестирования производительности)
- Соберите и запустите проект
## 📌 Текущее состояние

Сейчас движок находится в активной разработке.

Основной фокус:

- стабилизация архитектуры
